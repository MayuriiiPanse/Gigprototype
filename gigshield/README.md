# GigShield - Financial Identity OS for Gig Workers

Modern Vite + React + Tailwind version with Gemini AI features.

## Setup
1. `npm install`
2. Copy `.env.example` to `.env` and add your Gemini key: `VITE_GEMINI_API_KEY=...`
3. `npm run dev`

## Build & Deploy
`npm run build` → upload `dist` folder to Vercel/Netlify.

- 📊 Dashboard with earnings charts
